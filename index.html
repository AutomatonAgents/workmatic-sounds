How it works

From a track to an instrument

01

Every note becomes a sample

The song is separated into stems and cut at every note event. Each occurrence keeps its exact original audio — nothing is re-synthesized.

02

Compiled into real VSTs

The slices are mapped into our own compiled instruments (lead, 808, drums, vox) with die-out tails and loudness-matched keys for live playing.

03

Pure MIDI, on the grid

The included FL Studio project reproduces the track from 4-bar patterns whose notes sit exactly on the lines — editable like any beat you'd write yourself.

FAQ

Questions

What exactly is in a pack?

Compiled VST instruments (one per stem: lead, 808, drums, vox), the per-occurrence audio they stream, an FL Studio project that plays the full song from pure MIDI, and an installer that sets everything up in one run.

Which DAWs are supported?

The instruments are 64-bit Windows VST2 plugins — they load in FL Studio, Ableton Live, Reaper and most Windows hosts. The included project file is FL Studio 2025.

Can I change the notes?

Yes — that's the point. The project is ordinary MIDI in ordinary patterns. Move, mute and rearrange freely. One rule: don't edit note velocities — velocity selects which original audio slice plays.

How do I install?

Unzip, run INSTALL.bat (as administrator, or pass your own VST folder), rescan plugins in your DAW, open the included .flp, press play.
