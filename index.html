FAQ

Questions

What exactly is in a pack?

Four VST instruments (lead, 808, drums, vox), an FL Studio project that plays the full song, and an installer that sets everything up in one run.

Which DAWs are supported?

The instruments are 64-bit Windows VST2 plugins — they load in FL Studio, Ableton Live, Reaper and most Windows hosts. The included project file is FL Studio 2025.

Can I make my own music with them?

Yes — the project is fully editable and every instrument is playable like any VST. Turn on LIVE PLAY on the plugin for even hand playing. One tip: leave note velocities in the included project as they are for the authentic sound.

How do I install?

Unzip, run INSTALL.bat (as administrator, or pass your own VST folder), rescan plugins in your DAW, open the included .flp, press play.
