# Workmatic Sounds

Songs rebuilt as playable VST instruments — every note is the original audio,
every FL Studio project is pure MIDI on the grid.

Each pack contains compiled VST instruments (HISE), the per-occurrence audio
they stream, an `INSTALL.bat` that sets everything up on any Windows machine,
and an open-and-play `.flp`.

Packs are published as GitHub Releases of this repository; the storefront
(`index.html`) is a static page deployable on GitHub Pages / Vercel as-is.

MIDI velocities are occurrence selectors — never edit them.
